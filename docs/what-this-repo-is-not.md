# What This Repo Is NOT

## Not the Clarion source repository

This repo does not contain the Clarion engine, CLI implementation, or any production code. It is a demo and documentation repository only.

## Not the full conformance harness

The trust-bite demo exercises one rule type (protected-file modification). The full conformance harness tests dozens of rule types, edge cases, and integration scenarios. That harness is internal.

## Not a replacement for `clarion trial`

The demo shows enforcement works. To evaluate Clarion on your actual codebase, run `clarion trial`. The demo fixture is intentionally minimal.

## Not a fake fixture pretending to be product state

The fixture in `fixtures/trust-bite-app/` is clearly labeled as a demo fixture. It does not pretend to be a real project or real governance state. It exists solely to demonstrate the enforcement mechanism.

## Not a marketing site

This repo contains technical documentation and working code. It does not contain promotional material, pricing, or sales content.
